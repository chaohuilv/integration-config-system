1. Maven 依赖

<dependency> <groupId>com.integration</groupId> <artifactId>integration-sdk</artifactId> <version>1.0.0</version> </dependency>

2. 快速开始

// 初始化客户端 IntegrationClient client = new IntegrationClient("http://localhost:8080"); // GET 请求（无参数） String result = client.invoke("user-list"); // GET 请求（带 URL 参数） String result = client.invoke("user-get", Map.of("id", 1)); // POST 请求（带参数和请求体） String result = client.invoke("user-create", Map.of("deptId", 10), // URL 参数 "{\"name\":\"张三\",\"age\":25}" // 请求体 ); // POST 请求（带自定义请求头） Map<String, String> headers = new HashMap<>(); headers.put("X-Request-Id", "12345"); headers.put("X-Client-Version", "1.0.0"); String result = client.invoke("user-create", Map.of(), // URL 参数 "{\"name\":\"张三\"}", // 请求体 headers // 自定义请求头 ); // 异步调用 CompletableFuture<String> future = client.invokeAsync("user-list", null, null, null); String result = future.get();

3. 完整示例

import com.integration.sdk.IntegrationClient; import java.util.Map; import java.util.HashMap; public class Example { public static void main(String[] args) { IntegrationClient client = new IntegrationClient("http://localhost:8080"); try { // 示例1：简单 GET 请求 String userJson = client.invoke("user-get", Map.of("id", 1001)); System.out.println("用户信息: " + userJson); // 示例2：POST 请求，带自定义请求头 Map<String, String> headers = new HashMap<>(); headers.put("X-Request-Id", "req-" + System.currentTimeMillis()); headers.put("Accept-Language", "zh-CN"); String newUserJson = client.invoke("user-create", Map.of("deptId", 10), "{\"username\":\"zhangsan\",\"email\":\"zhangsan@example.com\"}", headers ); System.out.println("创建结果: " + newUserJson); // 示例3：覆盖配置中的认证头（调试用） Map<String, String> authHeaders = new HashMap<>(); authHeaders.put("Authorization", "Bearer test-token-123"); String result = client.invoke("api-test", Map.of(), null, authHeaders); } catch (IntegrationException e) { System.err.println("调用失败: " + e.getMessage()); } } }

4. Header 传参示例

当目标接口需要通过请求头传递认证信息（如 API Key、Token）时，按以下步骤配置：

步骤一：在接口配置中设置认证方式

📝 接口配置页面填写示例：

认证类型API_KEY
认证位置HEADER（请求头）
认证参数名X-API-Key
认证值your-secret-api-key-here

💡 系统会自动在请求头中添加：X-API-Key: your-secret-api-key-here

步骤二：常见的 Header 认证方式

🔹 API Key 认证

认证类型: API_KEY
认证位置: HEADER
认证参数名: X-API-Key
认证值: sk-xxxxxxxxxxxxxx

→ 请求头: X-API-Key: sk-xxxxxxxxxxxxxx

🔹 Bearer Token 认证（OAuth 2.0）

认证类型: BEARER
认证位置: HEADER
认证参数名: Authorization
认证值: Bearer eyJhbGciOiJIUzI1NiIs...

→ 请求头: Authorization: Bearer eyJhbGciOiJIUzI1NiIs...

🔹 Basic Auth 认证

认证类型: BASIC
认证位置: HEADER
认证参数名: Authorization
认证值: Basic base64(username:password)

→ 请求头: Authorization: Basic dXNlcjpwYXNz

🔹 自定义 Header（多个请求头）

在"请求头"字段中填写（每行一个）：
Content-Type: application/json
X-Request-Id: {traceId}
X-Client-Version: 1.0.0

→ 请求时会自动携带以上请求头

步骤三：动态 Token 认证（自动获取Token）

⚡ 适用场景：Token 有过期时间，需要定期刷新

认证类型DYNAMIC
Token接口编码auth-token-get（系统中已配置的获取Token接口）
Token提取路径$.data.accessToken（从Token接口响应中提取）
Token参数位置HEADER
Token请求头参数名Authorization
Token前缀Bearer
Token缓存时间3600 秒（1小时）

💡 系统会自动调用 auth-token-get 接口获取Token，缓存后用于后续请求

5. SDK 方法

方法说明
invoke(apiCode)GET 请求，无参数
invoke(apiCode, params)GET/POST 请求，带 URL 参数
invoke(apiCode, params, body)POST/PUT 请求，带参数和请求体
invoke(apiCode, params, body, headers)完整调用，带自定义请求头（会与配置合并，动态值优先）
invokeAsync(...)异步调用，返回 CompletableFuture
batchInvoke(items)批量调用

6. Headers 参数说明

📋 headers 参数优先级规则：

配置表请求头接口配置中"请求头"字段的内容
认证请求头根据认证类型自动添加（Bearer / Basic / API Key）
动态请求头SDK 调用时传入的 headers 参数，优先级最高，会覆盖同名配置
// 示例：动态覆盖认证头（用于调试或切换账号）
Map<String, String> headers = new HashMap<>();
headers.put("Authorization", "Bearer another-token-xxx");
client.invoke("user-get", Map.of("id", 1), null, headers);
// 此时使用的是 another-token-xxx，而非配置表中的 Token

7. C# (.NET) SDK

7.1 安装

# via NuGet Package Manager Install-Package IntegrationSDK # via .NET CLI dotnet add package IntegrationSDK

7.2 快速开始

// 初始化客户端 var client = new IntegrationClient("http://localhost:8080"); // GET 请求（无参数） string result = client.Invoke("user-list"); // GET 请求（带 URL 参数） string result = client.Invoke("user-get", new Dictionary<string, object> { { "id", 1001 } }); // POST 请求（带请求体） string result = client.Invoke("user-create", new Dictionary<string, object> { { "deptId", 10 } }, // URL 参数 "{\"username\":\"zhangsan\",\"email\":\"zhangsan@example.com\"}" // JSON 请求体 ); // 带自定义请求头 var headers = new Dictionary<string, string> { { "X-Request-Id", "req-" + DateTimeOffset.UtcNow.ToUnixTimeMilliseconds() }, { "Accept-Language", "zh-CN" } }; string result = client.Invoke("user-create", new Dictionary<string, object>(), "{\"name\":\"张三\"}", headers );

7.3 完整示例

using IntegrationSDK; using System; using System.Collections.Generic; using System.Threading.Tasks; public class Example { public static async Task Main(string[] args) { var client = new IntegrationClient("http://localhost:8080"); try { // 示例1：简单 GET 请求 string userJson = client.Invoke("user-get", new Dictionary<string, object> { { "id", 1001 } }); Console.WriteLine("用户信息: " + userJson); // 示例2：POST 请求 string newUserJson = client.Invoke("user-create", new Dictionary<string, object> { { "deptId", 10 } }, "{\"username\":\"zhangsan\",\"email\":\"zhangsan@example.com\"}"); Console.WriteLine("创建结果: " + newUserJson); // 示例3：自定义请求头覆盖认证 var authHeaders = new Dictionary<string, string> { { "Authorization", "Bearer test-token-123" } }; string result = client.Invoke("api-test", new Dictionary<string, object>(), null, authHeaders); // 示例4：异步调用 string asyncResult = await client.InvokeAsync("user-list"); Console.WriteLine("异步结果: " + asyncResult); } catch (IntegrationException ex) { Console.Error.WriteLine("调用失败: " + ex.Message); } } }

8.4 Header 认证示例

# API Key 认证 headers = { "X-API-Key": "sk-xxxxxxxxxxxxxx" } result = client.invoke("api-test", headers=headers) # Bearer Token 认证 headers = { "Authorization": "Bearer eyJhbGciOiJIUzI1NiIs..." } result = client.invoke("api-test", headers=headers) # Basic Auth 认证 import base64 credentials = base64.b64encode(b"username:password").decode("utf-8") headers = { "Authorization": f"Basic {credentials}" } result = client.invoke("api-test", headers=headers) # 多个自定义 Header headers = { "Content-Type": "application/json", "X-Request-Id": "req-12345", "X-Client-Version": "1.0.0" } result = client.invoke("api-test", headers=headers)

📥 导入接口

支持 curl / OpenAPI / Swagger 多格式导入，以及 OpenAPI 3.0 导出

📋 粘贴 curl 命令

💡 支持的格式： 支持单行和多行 curl 命令，自动识别 GET/POST/PUT/DELETE/PATCH 方法，Bearer Token / Basic Auth / API Key 认证方式

📄 粘贴 OpenAPI / Swagger JSON

💡 支持的格式： OpenAPI 3.0 JSON 和 Swagger 2.0 JSON。支持 operationId 作为接口编码，summary 作为名称，tags[0] 作为分组，query 参数自动解析为 JSON

📤 导出为 OpenAPI 3.0 JSON

💡 导出说明： 导出全部 启用状态 的接口，生成标准 OpenAPI 3.0.3 JSON 文件，可直接导入 Apifox、Postman、Apipost 等工具
